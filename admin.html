📌 How to update the live site for all customers:
  1. Click "Generate & Download data.js" below
  2. Go to your GitHub repository
  3. Navigate to js/data.js
  4. Click Edit (pencil icon) → paste the new content → Commit
  5. Your site updates within minutes! ✅
