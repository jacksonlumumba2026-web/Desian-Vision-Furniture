🐙 GitHub Integration — Auto-Publish Photos & Products
How it works: Enter your GitHub token below. Once connected, uploading a photo AND saving a product (or price/special change) both go directly to your GitHub repo and become live on your website for all customers automatically — no manual export/paste needed. Without this connected, changes are only saved on this device/browser and won't appear for other visitors.

To get a GitHub Token:
  1. Go to GitHub.com → Your Profile → Settings
  2. Click Developer settings → Personal access tokens → Tokens (classic)
  3. Click "Generate new token"
  4. Give it a name (e.g. "DVF Admin"), tick "repo" scope
  5. Click Generate and paste the token below
👁️
Your token is saved only on this device and never shared.
Use this right after connecting GitHub for the first time — it pushes every product, price and photo already saved on this device straight to the live site, instead of waiting for the next individual edit.
